# 事前登録: K殻蛍光X線実装のEGS5相互検証

作成日: 2026-07-18（この時刻のChatCarlo/EGS5双方の実行結果は未取得。以下は
結果を見る前に固定する）

## 検証したい実装

[docs/plan_fluorescence.md](../../plan_fluorescence.md)で実装したK殻蛍光X線
（コミット b732d35 / a6f133c）。ChatCarloはK殻のみ実装、L殻・カスケードは
非目標として明記済み。

## 幾何・線質

- 単色100 keV光子の鉛筆ビームが鉛(Pb)スラブに垂直入射（`tutor5`/`water60.f`と
  同じLATCH方式の踏襲、材料のみ鉛に変更）
- スラブ厚さ: 0.05 cm（500 μm）。理由: `linear_mu(lead, 100keV) ≈ 63.0 /cm`
  （1/e深さ≈0.016cm）なので、この厚さは約3.15平均自由行程分。無衝突透過は
  少数だが、表面近傍で生じた蛍光光子が反対面から脱出するのに十分な統計が
  得られる厚さとして選定（`tests/test_fluorescence.py`の
  `test_escaped_spectrum_shows_fluorescence_peak`と同じ厚さ・同じ根拠）
- history数: 両コードとも1,000,000（統計誤差を評価量ごとに二項近似で算出）
- ChatCarlo seed=1、EGS5 inseed=1（既存の慣例を踏襲）
- 物理モデル: IBOUND=1（束縛コンプトン、既存の必須事項）・IRAYL=1
  （レイリー有効）を両コードで揃える。蛍光フラグ（EGS5側の正式名称・構文は
  egs5-operatorがPEGS5/EGS5マニュアルAppendix B/Cで確認し、記憶で断定しない）
  はON/OFFの2条件を作る

## 評価する量と許容基準（結果を見る前に固定）

### 観測量A（主）: 脱出光子の全エネルギー透過率

`Σ(脱出光子のエネルギー) / (入射光子数 × 100 keV)`。蛍光光子が持ち出す
エネルギーを含むかどうかで、ON/OFFの違いが最も直接的に現れる量。

- **ステップ1（対照実験）**: 蛍光OFF条件（ChatCarlo `physics.fluorescence:
  false` vs EGS5 蛍光無効）で比較。これは新規実装（蛍光）を含まない、
  既存の相互検証範囲内の量（束縛コンプトン+光電+レイリーの鉛内輸送、
  新材料・新エネルギーでの初検証）。
  **合格基準: 相対差2%以内かつ2σ以内**（Phase 1水60keVと同じ基準を踏襲）。
- **ステップ2（本題）**: 蛍光ON条件で比較。
  **合格基準: 相対差5%以内かつ3σ以内**（Phase 1より緩い基準にする理由:
  ChatCarloはK殻のみ・EGS5側がL殻蛍光も含む可能性があり、その差が
  観測量Aに系統的に効きうるため。緩めた分はこの理由を明記し、結果を見てから
  緩めたのではないことをこの文書で担保する）
- **ステップ3（差分の一致、最も重視する）**: 蛍光による変化量
  `Δ = (観測量A)_ON - (観測量A)_OFF` を両コードで計算し、比較する。
  蛍光という単一の物理過程が引き起こす変化そのものが両コードで同程度かを見る、
  「一度に1つの差だけを見る」設計。
  **合格基準: ΔのChatCarlo/EGS5比が0.7〜1.3倍の範囲内**（L殻分の寄与を
  許容する幅として、ステップ2の5%より広く取る。ΔはOFF条件の系統差の影響を
  ある程度打ち消すため、ステップ2よりむしろ厳しく解釈しうる指標だが、
  蛍光光子自体の統計揺らぎがOFF単体よりばらつくため幅を広げる）

### 観測量B（副・定性的）: 脱出光子スペクトルの蛍光ピーク帯(72–86 keV)の割合

`tests/test_fluorescence.py::test_escaped_spectrum_shows_fluorescence_peak`
と同じ帯域（Pb Kα2/Kα1/Kβ3/Kβ1: 72.80/74.97/84.45/84.94 keV相当）。

- **合格基準**: 蛍光ONでOFFの2倍以上に増える、という定性的傾向が
  **両コードで**再現されること（ChatCarlo単体では既に確認済み: OFF 8.6%→ON
  82.1%、n=200,000、厚さ0.05cm）。定量的な一致は求めない
  （EGS5がPb蛍光線を何本モデル化しているか・L殻を含むかで帯域内の内訳が
  変わりうるため。観測量Aのステップ3の方を主たる定量判定に使う）

## 想定される既知の系統差要因（結果を見る前に列挙）

- **L殻蛍光の非対称**: ChatCarloはK殻のみ、EGS5がIEDGFL相当のフラグで
  L殻も含めてモデル化していれば、ON条件でEGS5の方が透過エネルギーが
  やや高くなる方向のズレが予想される（鉛L線10.5–12.6 keVは鉛内平均自由行程が
  μmオーダーで、0.05cm厚のスラブでは大半が再吸収されるため、透過エネルギー
  への寄与は小さいと予想するが、確認前の推測に留める）
- **線エネルギー値の出典差**: ChatCarloはxraylib（EPDLベース）のLineEnergy/
  RadRateから4線（Kα2/Kα1/Kβ2/Kβ1相当）を使用。EGS5がICRU/ENDFなど別出典の
  テーブルを使っていれば、観測量Bのピーク帯の詳細な位置・本数が異なりうる
  （帯域を72-86 keVと広めに取ることで吸収）
- **K殻イオン化確率のモデル差**: ChatCarloは`CS_Photo_Partial/CS_Photo`
  （xraylib, EPDLベース）。EGS5が別の光電殻別断面積テーブルを使っていれば、
  蛍光発生率自体に数%の差が出る可能性がある（観測量Aの許容基準5%に織り込み済み）
- **密度**: 鉛密度はChatCarlo/xraylib基準で11.35 g/cm³固定
  （`materials.py: _DENSITY_OVERRIDE`）。EGS5 PEGS5入力のRHOをこれに揃える
  よう明記する（揃え忘れは既知の落とし穴、water60 crosscheckの教訓）

## 判定の扱い

- 上記基準に対して不合格が出ても基準そのものを緩めない。原因調査を別項目として
  記録し、未解決のまま「条件付き合格」として残すことを許容する
  （前例: Phase 2b PDD、`docs/egs5_crosscheck/PDD_RESULTS.md`）
- 観測量Bは定性的判定のみで、観測量Aステップ3を最終合否の主指標とする
